Recent Work

Food or Not Food Text Classifier

Built a binary text classifier that predicts whether a sentence is about food using transfer learning with Hugging Face Transformers. The project leverages a synthetic dataset generated with Mistral Chat/Mixtral and was developed with DistilBERT (uncased). Implemented AutoTokenizer for text encoding, fine-tuned the model, and optimized training through evaluation of accuracy and loss metrics. To ensure robustness, the pipeline was designed for scalable preprocessing and efficient inference. The model is deployed on Hugging Face Spaces with Gradio as an interactive demo, enabling real-time text classification.

Dog Breed Identification App

Developed a deep learning image classifier that predicts the breed of a dog from an input photo using transfer learning with TensorFlow 2 and MobileNetV2, based on the Kaggle Dog Breed Identification dataset. Built a complete pipeline from data preprocessing to model evaluation, implementing one-hot encoding, data augmentation, and optimized TensorFlow input pipelines with batching, caching, prefetching, and AUTOTUNE. Applied early stopping, log loss, classification reports, and confusion matrices to assess and improve generalization. The model is deployed via Streamlit on Hugging Face Spaces.
